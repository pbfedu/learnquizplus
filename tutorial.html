Instrucciones para generar el archivo .txt

Estas son las instrucciones detalladas para que cualquier IA pueda generar un archivo de preguntas compatible con la plataforma.

Cómo generar un archivo de preguntas para el Quiz Modular

Este documento explica el formato que debe seguir un archivo de texto (.txt) para ser compatible con la plataforma de quiz. Cada pregunta debe tener un formato específico y estar separada de la siguiente por una línea con tres guiones ---.

Formato General

Cada pregunta debe ser un bloque de texto que contenga pares de clave-valor. La clave y el valor deben estar separados por dos puntos y un espacio (: ).

clave_1: valor_1 clave_2: valor_2 ... ---

Tipos de Pregunta y sus Formatos

A continuación se detalla el formato para cada uno de los tipos de pregunta que la plataforma soporta.

Pregunta de Opción Múltiple (multiple-choice)

  • type: multiple-choice (siempre).
  • question: El texto de la pregunta.
  • options: Una lista de opciones, separadas por comas.
  • correctAnswer: La opción correcta, tal y como aparece en la lista.
type: multiple-choice question: Elige la traducción de 'dog' options: Gato, Perro, Pájaro correctAnswer: Perro

Pregunta de Construir la Frase (word-bank)

  • type: word-bank (siempre).
  • question: La frase que el usuario debe construir.
  • correctSentence: La frase correcta, con cada palabra separada por una coma.
  • wordBank: El banco de palabras disponible. Debe incluir todas las palabras de correctSentence y algunas palabras extra para confundir.
type: word-bank question: Construye la frase: 'The cat is black' correctSentence: The, cat, is, black wordBank: black, is, The, cat, a, blue, dog

Pregunta de Emparejar (match)

  • type: match (siempre).
  • question: El texto de la pregunta.
  • pairs: Pares de palabras a emparejar. Cada par se separa por un signo de igual (=) y los pares por un punto y coma (;).
type: match question: Empareja las palabras pairs: book=libro; house=casa; apple=manzana

Pregunta de Verdadero o Falso (true-false)

  • type: true-false (siempre).
  • question: La afirmación a evaluar.
  • correctAnswer: La respuesta correcta, que debe ser true o false (en minúsculas).
type: true-false question: La frase 'I am a boy' significa 'Soy una niña' correctAnswer: false

Pregunta de Deslizador (slider)

  • type: slider (siempre).
  • question: El texto de la pregunta que pide un número.
  • min: El valor mínimo del deslizador (un número entero).
  • max: El valor máximo del deslizador (un número entero).
  • correctAnswer: El valor numérico correcto.
type: slider question: ¿Cuántos años tiene el búho Duo? min: 0 max: 20 correctAnswer: 10

Pregunta de Rellenar el Espacio en Blanco (fill-blank)

  • type: fill-blank (siempre).
  • question: La frase con un espacio en blanco, representado por ___.
  • options: Las posibles palabras para rellenar el espacio, separadas por comas.
  • correctAnswer: La palabra correcta.
type: fill-blank question: Completa la frase: 'I like to ___' options: run, learn, eat, sleep correctAnswer: learn

Puntos Importantes a Recordar:

  • Separación: Cada pregunta debe estar separada por una línea de tres guiones (---).
  • Espacios: Asegúrate de que haya un espacio después de los dos puntos (: ).
  • Valores sin espacios iniciales/finales: Los valores se limpian automáticamente, pero es mejor que no tengan espacios extra.
  • Coherencia: El correctAnswer de una pregunta de opción múltiple debe coincidir exactamente con una de las options.
